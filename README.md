# Todo List App

<p>To show the live demo <a href="https://todo-list-app-teal-tau.vercel.app/">Click Here</a></p>

The **Todo List App** is a simple and interactive task management application built with **React**.  
It allows users to **add, complete, and delete tasks**, with all data stored in the browser using **Local Storage**.

## 🛠 Technologies Used

- **React.js** – Frontend framework
- **JavaScript (ES6+)** – Logic and state management
- **Tailwind CSS / DaisyUI** – Styling
- **Lucide React** – Icons
- **Local Storage** – Data persistence

## 🚀 Features

- Add new tasks
- Mark tasks as completed / incomplete
- Delete tasks
- Persistent data using Local Storage
- Clean and responsive UI
- Real-time UI updates
- Toggle task status with a click
- Icons for better user experience

## 🎯 Learning Purpose

This project was built to practice:

- React Hooks (`useState`, `useEffect`, `useRef`)
- Component-based architecture
- State management
- Event handling
- Local Storage integration
- Conditional rendering
- Clean UI design
